UserCandy Framework Extenders

Extenders are used to load any code within given parts of the UserCandy Framework.

The file name determains where the file is included within the site.

For example ext.index.top.php
ext - lets the framework know the file is an extender file.
index - lets the framework know the file is loaded within the index file.
top - lets the framework know the file is loaded at the top of the file.
php - required file extension to be able to be included.

These files can contain any code you desire.  Can be used to display data,
record data to the database, add security to a given file, and much more.

The following file names are currently supported in the UserCandy Framework:

Note: These are case sensitive.
Remember: Always make a backup before editing.
